# Collocate - Business Management Platform

![JavaFX](https://img.shields.io/badge/JavaFX-17+-blue.svg) ![Firebase](https://img.shields.io/badge/Firebase-Cloud%20Backend-orange.svg) ![Gemini API](https://img.shields.io/badge/Gemini%20API-AI%20Powered-blueviolet.svg)

Collocate is an all-in-one business management platform designed to help freelancers and small businesses organize their projects, finances, and client relationships in a unified desktop application.



## About The Project

Built with JavaFX, Collocate provides a centralized dashboard where users can manage their entire workflow, from tracking project progress and deadlines to generating invoices and logging expenses. By integrating external applications and leveraging the Gemini API for AI-powered assistance, the platform aims to unify the user's digital workspace, reduce administrative overhead, and allow business owners to focus on their core services.

## Key Features

* **Centralized Dashboard**: Get a high-level overview of your business activities at a glance.
* **Client & Project Management**: Full CRUD (Create, Read, Update, Delete) functionality for managing clients and tracking project status, deadlines, and budgets.
* **Interactive Calendar**: Schedule appointments and milestones, and link them directly to clients or projects.
* **Finance & Invoicing**: Easily generate professional invoices, track billable hours, and log business expenses.
* **AI Assistant**: Utilize the Gemini API to draft emails, get smart reminders for deadlines, and summarize project progress.
* **External Integrations**: Securely sync data with third-party applications like Google Calendar and Google Docs.

## Technology Stack

* **Frontend**: JavaFX, FXML, CSS
* **Backend & Database**: Firebase (Cloud Firestore)
* **AI Integration**: Google Gemini API
* **UI Design**: Scene Builder
